\# Fallout 4 ESM Patcher



\## Version 1.1.0



A standalone tool to patch VR/1.10.163/Next-Gen Fallout4.esm files for compatibility. This will ddelta patch your Fallout4.esm to a compatible format. Supported conversions are:

VR to 1.10.163/NG
1.10.163 to NG
NG to 1.10.163
